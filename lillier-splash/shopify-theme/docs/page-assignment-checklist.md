# Lillier Required Pages and Template Assignment

Use this checklist in Shopify Admin after uploading the theme.

## Page records to create in `Online Store -> Pages`

1. `About`  
   Handle: `about`  
   Template: `about` (`templates/page.about.json`)

2. `Brands`  
   Handle: `brands`  
   Template: `brands` (`templates/page.brands.json`)

3. `Stories`  
   Handle: `stories`  
   Template: `stories` (`templates/page.stories.json`)

4. `Contact`  
   Handle: `contact`  
   Template: `contact` (`templates/page.contact.json`)

5. `Splash` (optional if you keep a transition splash in this theme)  
   Handle: `splash`  
   Template: `splash` (`templates/page.splash.json`)

## Auto-routed templates (no page record assignment needed)

- Homepage: `templates/index.json`
- Product pages: `templates/product.json`
- Collection pages: `templates/collection.json`
- All collections index: `templates/list-collections.json`
- Cart: `templates/cart.json`
- Search: `templates/search.json`
- Blog index: `templates/blog.json`
- Blog article: `templates/article.json`
- 404: `templates/404.json`
